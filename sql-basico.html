SQL
BÁSICO

Este curso es el punto de partida ideal si nunca escribiste una consulta SQL o querés consolidar las bases. Aprendés desde cero, con ejemplos reales y un ritmo pensado para que nada quede en el aire.

Al finalizar vas a poder leer, escribir y entender consultas SQL con confianza, filtrar y ordenar datos, y trabajar con tablas relacionadas usando JOINs.

QUÉ VAS A APRENDER

  • → Fundamentos Qué es una base de datos, cómo funciona y cuándo usar SQL o NoSQL
  • → Modelado de Datos Creación de tablas, tipos de datos, claves primarias y foráneas
  • → Manipulación de Datos INSERT, UPDATE y DELETE para trabajar con información real
  • → Consultas SQL SELECT, WHERE, ORDER BY, GROUP BY
  • → Funciones COUNT, SUM, AVG, MIN, MAX
  • → Filtros y Condiciones LIKE, IN, BETWEEN, operadores lógicos
  • → Relaciones Modelos 1-N e INNER JOIN aplicado a casos reales
